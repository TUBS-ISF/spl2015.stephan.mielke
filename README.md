# spl2015.stephan.mielke
========================

Options:
--------
- --client, -c
 - start client
 - Default: true
- --commands
 - supported commands (client) example: "nick" "names" "get"
 - Default: []
- --database, -db
 - start server with database
 - Default: false
- --debug, -d
 - Debug mode
 - Default: false
- --help, -h
 - Default: false
- --server, -s
 - start server
 - Default: false
